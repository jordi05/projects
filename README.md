# Toolkit de Redes y Seguridad

Herramienta web con tres utilidades para estudio rapido: calculadora de subredes, analizador de contrasenas y limpiador de logs con regex. Todo funciona en el navegador.

## Demo

- https://netlabtools.org

## Funcionalidades

- Calculo de red, broadcast, rango util y hosts.
- Analisis de robustez de contrasenas con entropia estimada.
- Filtro y limpieza de logs con regex, mas snippet JS/Python.
- Exportacion a PDF y PNG.
- Modo claro/oscuro y ES/EN.

## Desarrollo local

1. Abre index.html en el navegador.
2. Opcional: usa un servidor estatico simple (por ejemplo, python -m http.server).

## Deploy en GitHub Pages

- Settings > Pages
- Source: Deploy from a branch
- Branch: main / (root)
